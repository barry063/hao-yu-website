# Contributing

This repository is primarily a **personal academic website**. Day-to-day updates (CV, publications, copy) are made by the author.

## Forking for your own site

You are welcome to **fork** the repo and adapt it:

1. Replace biographical content in `index.html` with your own factual information.
2. Update `styles.css` if you want different typography or colours (CSS variables are at the top of the file).
3. Change `LICENSE` copyright if you redistribute a derivative; keep MIT if you reuse the layout substantially.
4. Replace `assets/og-image.png` with your own 1200×630 preview image for social sharing.
5. Set your GitHub Pages URL in `index.html` (`canonical`, `og:url`, `og:image`).

Please do not submit PRs that change the author’s personal CV data unless you are explicitly asked.

## Issues

Bug reports for the **template** (broken layout, accessibility, typos in boilerplate) are welcome via Issues.
